# Text-Editor
Python Text Editor using Tkinter with tab functionality.

Made using Python 3.4.3 on Windows

![picture alt](https://i.imgur.com/I3Sb2bL.png "Notepad+=1")

## Menus & Functionality:

* ### File:  
  * New  
  * Open  
  * Save As...  
  * Close  
  * Exit  

* ### Edit and Right click in text window:  
    * Undo  
    * Cut  
    * Copy  
    * Paste  
    * Delete  
    * Select All  
    
* ### Format:  
    * Word Wrap  
    
* ### Tabs:  
    * Right Click  
      * New Tab Option  
    * Center Click  
      * Close Tab  
    * Drag  
      * Rearrange Tab  
      
* ### Keyboard Shortcuts:  
    * Ctrl-s: Save  
    * Ctrl-o: Open  
    * Ctrl-w: Close tab  
    * Ctrl-n: New tab  
    * Ctrl-tab: Next tab  
    * Ctrl-shift-tab: Previous tab  
    * Ctrl-u: Undo  
    * Ctrl-y: Redo
